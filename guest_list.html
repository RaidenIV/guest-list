Guest List Generator

How to Use the Guest List Generator

Show Details

Fill in the show name, headliner, direct support, date, and day-of-show ticket pricing. The show name and artist names are auto-capitalized when you leave the field.

Guest Lists

Enter one name per line in each guest list box. Names are automatically sorted alphabetically and capitalized. Duplicates across any list are flagged in red.

  • Headliner / Direct Support — artist guest lists
  • XODIA — house / company guests
  • Free Spots — sponsors, comps, venue holds, or special approvals
  • Promo Team — promotional staff

DJs

Set how many DJs are performing. For each DJ, choose Solo, Duo, or Trio, then enter stage name(s), legal name(s), and that DJ’s guest list. DJ blocks can be dragged to reorder them.

Other Sections

Add custom sections such as VIP, Press, Venue Staff, Family, or Management. Name the section, add one guest per line, and drag sections to reorder them.

Preview

The right panel shows a live preview of the print-ready guest list. If the list spans multiple pages, each page appears as a separate sheet and a multi-page warning appears in the preview controls.

Buttons

  • Download PDF — download the print-ready guest list PDF
  • Copy — copy the full guest list as plain text; the button changes to “Copied!” after a successful copy
  • Download CSV — download the current guest list as a local CSV file
  • Load — load a saved CSV or legacy JSON file from your computer
  • Save — save the current guest list to MongoDB; duplicate saved-list names are blocked
  • Reset — clear the page immediately

Saved Guest Lists

When the backend is connected, the saved guest list dropdown fills with MongoDB records. Selecting a guest list automatically loads it into the generator.

Status Messages

The status indicator reports server connection, save success, load success, duplicate-name warnings, and backend errors without interrupting your workflow.

Import by Drag & Drop

Drag a saved CSV or JSON file from your computer and drop it anywhere on the preview panel to import it instantly.
